# Changelog

## v0.1.0

Initial public release.

- Route decomposed execution tasks to matching `SKILL.md` workflows.
- Support single-task and batch JSON output.
- Scan multiple local skill directories.
- Merge optional community mappings into installed skills.
- Surface missing useful skills with install hints.
- Provide macOS, Windows PowerShell, Linux, and generic shell installers.
- Record local recommendation audit events for hit-rate review.
- Summarize recommendation hit rate with `--stats`.
- Include unit tests and cross-platform CI.
